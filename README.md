# Projet Agenda — C++

Un agenda complet en C++ permettant de gérer des événements, de modifier un agenda, de sauvegarder les données, d’exporter en HTML et JSON, ainsi que d’importer des fichiers JSON.

Ce projet a été réalisé dans le cadre du BUT Informatique.

---

## Fonctionnalités

### Gestion de l’agenda
- Modifier le titre et la description de l’agenda
- Afficher l’agenda (description + événements)

### Gestion des événements
- Ajouter un événement
- Supprimer un événement
- Génération automatique d’un identifiant unique
- Vérification de validité des dates
- Vérification que la date de fin est après la date de début

### Export / Import
- Export HTML (`agenda.html`)
- Export JSON (`*.json`)
- Import JSON (titre importé, structure extensible)

### Formats supportés
- `.txt`
- `.html`
- `.json`
